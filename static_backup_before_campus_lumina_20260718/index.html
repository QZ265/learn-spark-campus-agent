Multi-Agent

不是单一聊天框，而是学习任务流水线

星辰工作流按任务调用三个真实角色；本地知识检索和 SQLite 存储作为可验证的工具层，不冒充 Agent。

ProfileAgent

只从对话和练习中的直接证据抽取画像，记录来源、置信度、冲突和变化。

ResourceAgent

结合画像证据、课程知识片段和允许引用，生成五类结构化资源。

ReviewAgent

核对引用支持关系、编造风险和 Python 代码，失败时触发重新生成。

Resources

一次对话生成五类可打开资源

轻量版先聚焦 Python 入门课程，后续可把知识库扩展到其他专业课、通识课或校园服务。

专业课程讲解文档 知识点思维导图 分层练习题库 拓展阅读材料 代码实操案例
Learning Loop

从答疑到评估，再回到个性化推荐

学生提交练习答案后，系统会给分、指出细节问题，并把结果写回画像，形成持续优化的学习闭环。

1. 提问学生用自然语言描述困难
2. 生成系统输出资源和学习路径
3. 练习学生提交代码或文字答案
4. 优化更新易错点和资源推送策略
